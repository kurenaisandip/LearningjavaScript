A Web API is a developer's dream. It can extend the functionality of the browser It can greatly simplify complex functions It can provide easy syntax to complex code What is Web API? API stands for Application Programming Interface. A Web API is an application programming interface for the Web. A Browser API can extend the functionality of a web browser. A Server API can extend the functionality of a web server. Browser APIs All browsers have a set of built-in Web APIs to support complex operations, and to help accessing data.
